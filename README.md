### Installation (Kali Linux)

```
$ git clone https://github.com/MohmmadALbaqer/DDoSAttack.git
$ cd DDoSAttack
$ pip3 install -r requirements.txt
$ python3 install.py
# OR
$ bash install.sh
$ python3 DDoSAttack.py

```
